# Portafolio Tecnológico

## Descripción
Portafolio digital profesional desarrollado para presentar proyectos académicos y evidencias de aprendizaje.

## Contenido
- Información personal
- Publicaciones de proyectos
- Habilidades técnicas
- Información de contacto

## Tecnologías utilizadas
- HTML5
- CSS3
- GitHub

## Objetivo
Fortalecer la marca personal tecnológica mediante la publicación organizada de proyectos académicos.
